# To-Do List com React e API

Este projeto é uma aplicação de lista de tarefas (To-Do List) que consiste em um front-end desenvolvido em React e um back-end que fornece uma API para operações de CRUD Completo das tarefas.

## Funcionalidades

- **Adicionar Tarefas:** Permite ao usuário criar novas tarefas.
- **Listar Tarefas:** Exibe todas as tarefas cadastradas.
- **Remover Tarefas:** Permite a exclusão de tarefas.
- **Marcar como Concluída:** Opção para marcar tarefas como concluídas ou pendentes.

## Tecnologias Utilizadas

- **Front-end:**
  - React
  - Chackra UI
  - HTML5
  - CSS3
  - JavaScript (ES6+)

- **Back-end:**
  - Node.js
  - ExpresS
  - MySQL
